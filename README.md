# DevLens

DevLens is an AI-powered web application that helps developers understand and improve codebases.

## Tech Stack

- Frontend: Next.js + TypeScript + Tailwind CSS
- Backend: FastAPI + Python
- Database: PostgreSQL
- Containers: Docker + Docker Compose
